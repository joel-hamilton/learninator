# SRS (Spaced Repetition System) for quiz content

**What**: Surface past quiz questions on a spaced repetition schedule to build long-term retention.

**Why**: The teach skill emphasizes storage strength over fluency. SRS directly implements "spacing" and "retrieval practice" from the teaching philosophy.

**Acceptance criteria**:
- [ ] Lessons can register quiz questions into an SRS queue
- [ ] Dashboard shows "X items to review today"
- [ ] Review page presents quiz items with timing based on past performance
- [ ] Track per-item accuracy and adjust intervals

**Notes**: SM-2 algorithm is the classic approach. May want to store quiz items from lesson HTML somehow.
