# Edit mission details

**What**: Allow users to edit their mission's title, goals, constraints, and out-of-scope items.

**Why**: The teach skill explicitly says "Missions may change as the user develops more skills and knowledge." Currently the mission is set during onboarding and can't be edited through the UI (only via AI chat).

**Acceptance criteria**:
- [ ] Edit button on mission workspace that opens inline form
- [ ] Editable fields: title, why, success criteria, constraints, out of scope
- [ ] Changes create a learning record capturing the mission shift
- [ ] AI is notified of the change (or user can also use chat)

**Notes**: The AI can already update mission content via tools. This is about giving users direct agency.
