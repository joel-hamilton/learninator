# Lesson re-generation and iteration

**What**: After leaving feedback on a lesson, allow the AI to regenerate it based on that feedback.

**Why**: The feedback loop is currently one-directional (user rates, that's it). The teach skill is designed for iterative teaching — if a lesson is too hard, the AI should be able to create a bridging lesson or simplify it.

**Acceptance criteria**:
- [ ] "Too hard" / "Too easy" feedback offers to regenerate the lesson
- [ ] AI creates a new lesson (or updates existing) addressing the feedback
- [ ] Feedback patterns are recorded in NOTES.md for future lesson design

**Notes**: Could also generate a "bridging lesson" between the current one and the next, rather than replacing.
