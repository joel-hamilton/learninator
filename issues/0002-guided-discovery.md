# Guided mission discovery quiz

**What**: Replace/augment the free-form "What do you want to learn?" with an interactive quiz that narrows down interests and suggests missions.

**Why**: Users often don't know exactly what they want to learn. A guided discovery flow (clickable options, branching questions) would help users who arrive without a specific goal.

**Acceptance criteria**:
- [ ] First-time user flow offers "guided discovery" option alongside free-form input
- [ ] Multi-step quiz asks about interests, goals, time commitment
- [ ] AI synthesizes quiz answers into a mission proposal
- [ ] User can accept, refine, or reject the proposal

**Notes**: N+1 feature. Free-form onboarding works for now.
